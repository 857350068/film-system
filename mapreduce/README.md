# MapReduce数据分析模块

该目录包含所有MapReduce任务的实现：

## 分析任务
1. **评价次数分析** - 计算评价次数最多的10部电影
2. **性别维度分析** - 计算不同性别评分最高的10部电影
3. **年龄段分析** - 计算各年龄段的平均电影评分
4. **电影类型分析** - 计算各类型电影评价最高的5部电影
